Simple test tool for PWM controlled FANs. 

(especially for the "HpFanControl" using Arduino Nano


Rotary encoder on pins A2, A3 (no external pull-ups or caps)
HD44780 LCD 16x2 on RS=D12, EN=D11, DB4-7 = D5, D4, D3, D2
PWM output on D9

Required libraries:
 - TimerOne
 - RotaryEncoder
 - HD44780
